Help & Data Formats

Importing data

Click the ⚙ Settings button and use Samsung Health import to upload either:

  • A ZIP export from the Samsung Health app (all metrics imported at once)
  • An individual Samsung CSV file (filename must start with the Samsung package prefix)
  • A plain CSV file — metric is detected automatically from the column headers

Re-importing the same file is safe — duplicates are skipped automatically.

Plain CSV format

MetricRequired columnsOptional columns
Blood pressure measured_at, systolic, diastolic pulse, notes
Weight measured_at, value_kg notes
Steps step_date, step_count distance_m

measured_at: YYYY-MM-DD HH:MM:SS or YYYY-MM-DDTHH:MM:SS
step_date: YYYY-MM-DD
distance_m: distance in metres

Weight goal

The weight goal is saved to your account on the server and is available on all devices. Set it in the Weight tab under the entry form.

Time range selector

Use the 1W / 1M / 3M / 6M / 1Y / 5Y / All buttons at the top of each data tab to filter the chart and table. The full dataset is always kept — filtering is display-only.
